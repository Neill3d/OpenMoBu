
	BlendShape Tookit

	Tools for creating corrective blendshapes for Models.

BlendShape Toolkit (suit with 3 elements) - 3 projects!

1) blendshapes tool - manage your blendshapes
2) Sculpting brush - modify your mesh
3) polygon tool - make snapshot, calculate delta, combine meshes and some pro features
	polygon tool - python commands to connect with some scripted dialog or to connect to work pipeline
4) deformer constraint - helper constraint to sculpt deformated meshes in real-time

License for these nodes, support for MotionBuilder's
